# Log Analyzer

Инструмент для анализа логов сетевых устройств и генерации отчетов по состоянию оборудования.

## Описание

Этот скрипт анализирует лог-файл сетевых устройств (Juniper MX, ACX) и создает детализированные отчеты по:
- Общему состоянию устройств
- Состоянию вентиляторов
- Температуре компонентов

## Зависимости

- pandas
- openpyxl
- python 3.12+

## Использование
Исходный лог-файл с именем result_output  
Запустите скрипт: python log_metro.py  
Отчет будет сгенерирован в формате Excel с именем alarm_report_metro_YYYY-MM-DD.xlsx  


### Выходные данные  
clear_log.txt - очищенный лог-файл  
dict_all.txt - данные в формате словаря  
alarm_report_metro_YYYY-MM-DD.xlsx - сформированный отчёт
Отчет в Excel содержит 4 листа:  
- Общая информация - таблица со всеми устройствами и значениями параметров
- Статистика - показывает проблемы и количество устройств
- Вентиляторы - таблица устройств с отключенными вентиляторами
- Температура - таблица устройств с высокой температурой (по умолчанию > 50C)
